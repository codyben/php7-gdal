# PHP7 GDAL

## Dependencies
- GDAL-dev (>v2.0)
- PHPCPP (v2.1.4)

## Installation
- Requirements
-   - PHP7.0

## Building the extension
-   Clone this repository
-   ```cd php7-gdal && make```
-   If all goes well, you should then be able to do ```sh deploy.sh``` which will copy your extension to the PHP extension directory
